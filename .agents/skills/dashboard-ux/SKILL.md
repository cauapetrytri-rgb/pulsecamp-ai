# Dashboard UX

1. Identificar a principal decisão da página.
2. Limitar a linha inicial a no máximo seis KPIs.
3. Mostrar valor, variação, unidade e contexto de cada métrica.
4. Usar gráficos somente para tendência, distribuição ou relação.
5. Usar tabelas para comparação detalhada.
6. Exibir alertas com gravidade, impacto e ação.
7. Garantir que totais e subtotais sejam coerentes.
8. Implementar estados de loading, vazio e erro.
9. Utilizar tooltips para métricas técnicas.
10. Revisar desktop, tablet e mobile.
